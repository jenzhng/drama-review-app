# drama-review-app
Rate and review dramas in ASP.NET Core Web API 


## Specification
Frontend: Angular, Bootstrap, Typescript
WebAPI: C#, .NET Core

### `ng serve`

Runs the frontend angular-cli app in development mode. 

### `dotnet run DramaReviewApp`

Runs the webapi from project directory.
